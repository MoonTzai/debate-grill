辩论筑基・Debate-Coach

I am a debate coach trained on the complete Debate Universal Grammar (辩论筑基) system by 精靈Moon, using the grill-me interview method. Thanks to Matt Pocock for the pioneering interview format, and to Wang Boyuan for inspiring its application to debate.

⚠️ This skill is currently calibrated for Chinese-language debate only. It is not adapted for English debate, and mainstream English debate differs significantly from the general debate style popular in Chinese-language debate. English functionality is for entertainment and reference only. Free video courses on YouTube & Bilibili (search "辩论筑基" or "精靈Moon").

⚠️ The responses of this skill are based on AI learning from the Debate Universal Grammar courseware (slides, not video lectures). Deviations from the understanding and intent of the author are inevitable. For knowledge content, refer to the video lectures of the author. The session content of this skill is for REFERENCE ONLY.

  1. API Token Consumption: This tool embeds the complete Debate Universal Grammar knowledge system as a system prompt (~20,000+ tokens). Every request consumes significant tokens. Heavy use may rapidly exhaust your API balance.

  3. File Tampering Risk: If you obtained this HTML file from a non-primary source, there is risk of third-party tampering or malicious code injection. Please obtain it from the original distribution channel whenever possible.
  4. Data Privacy: Your conversation content will be sent directly to a third-party AI API. Do not enter sensitive personal information.

Alternative: If you use Claude Code, ChatGPT, Cursor, or other AI tools, you can directly load the plain SKILL.md file without entering an API Key, avoiding Key leak risks.
