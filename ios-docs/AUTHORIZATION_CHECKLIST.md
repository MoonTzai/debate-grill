# Authorization Checklist Before App Store Submission

- Written permission covering App Store distribution
- Written permission covering use of the Debate-Coach name, screenshots, and marketing copy
- Written permission covering the bundled knowledge content and prompt files
- Confirmation that the submitting Apple Developer account is an approved distributor
- Confirmation whether free release only is allowed, or whether future monetization is permitted
- Required attribution text for the app, listing, and exported files
